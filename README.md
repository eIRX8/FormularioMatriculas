# 🎓 Formulário de Matrícula Escolar

Sistema completo de matrícula com validações, máscaras, upload de foto e conformidade com a LGPD.

## ✨ Funcionalidades

- ✅ RA gerado automaticamente (ano + 6 dígitos)
- ✅ Máscaras para CPF, Data, Celular e CEP
- ✅ Busca automática de endereço via ViaCEP
- ✅ Upload e pré-visualização de foto do aluno
- ✅ Validação em tempo real (idade mínima 18 anos)
- ✅ Integração com Supabase
- ✅ Conformidade com a LGPD
- ✅ Design responsivo e moderno

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
- Supabase (Backend)
- ViaCEP API

## 📦 Estrutura do Banco de Dados

- UnidadesEscolares
- Cursos
- Turmas
- Paises
- Matriculas

## 🚀 Como Executar

1. Clone o repositório
2. Configure as credenciais do Supabase no arquivo `index.html`
3. Execute os scripts SQL no Supabase
4. Abra o `index.html` no navegador
